# Changelog

All notable changes to this project will be documented in this file.

## [1.0.1] - 2026-02-16

### Initial Release
- txAdmin-compatible recipe installer for RedM servers on Linux.
- Supports downloading and verifying FXServer artifacts.
- Handles database setup (MySQL) and `server.cfg` generation.
- Includes `start.sh` generator for easy server management.
- Complete documentation: README, Contributing Guide, and Code of Conduct.
